# VI - Text editor
A brief cheatsheet for the VI text editor  

## Basics

    :	go to command mode
    :q	quit editor
    :q!	force quit editor
    :wq write and quit
      
    vim	NAME	open file 
    :edit NAME  open file 
    :e CTRL+D	show folder contents
    :Explore	open file explorer 
    
## Navigation
Use arrow keys or h,j,k,l to navigate
